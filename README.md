# Investimento2026

## Descrição
O projeto foi feito para poder simular um investimento, ou seja, ele vai ajudar o usuário
a ver quantas parcelas seram mais vantajosas para ele, qual o valor do investimento que dejesa pegar 
e qual seria a taxa de juros para aquele valor.

## Como Executar
1. Abra o Vs Code;
2. Procure a pasta "lib";
3. Dentro da pasta LIB clique em "main.dart";
4. Na barra embaixo procure por "No Device" ou "Windows *windows-x64";
5. Ao encontrar altere para "Chrome (web.javascript)";
6. Após alterar inicie o projeto no "Start Debugging" ou Play;
7. Se tiver realizado corretamente o projeto abrirá no chrome;
